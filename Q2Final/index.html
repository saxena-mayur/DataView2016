Part 1: Which states have done better than others?

Data Sets

  • H-DS-1: All India (from 2000 to 2011) and State-wise (2010 and 2011) number of cases and deaths due to specified diseases (Acute Diarrhoeal Diseases, Malaria, Acute Respiratory Infection, Japanese Encephalitis,Viral Hepatitis).

Methodology

  • Two possible scenarios to infer that a disease is controlled are:
    • Reduced numbers of cases
    • Increased cured cases count
  • For every disease, first we compute:
    • number of cured cases = number of cases - number of deaths
  • Next we compute a linear regression (y = mx+c) with x as year and y as number of cases and cured cases

Part 2: Which diseases have been better controlled than others in India?

Data Sets

  • F-DS-3: H-DS-1: All India (from 2000 to 2011) and State-wise (2010 and 2011) number of cases and deaths due to specified diseases(Acute Diarrhoeal Diseases, Malaria, Acute Respiratory Infection, Japanese Encephalitis, Viral Hepatitis), here Primary Census Abstract 2011 – India.

Assumption

  • Due to limited data, the progress has been monitored for two consecutive years i.e. 2010 and 2011

Methodology

  • Two possible scenarios to infer that a disease is controlled are:
    • Reduced numbers of cases
    • Increased cured cases count
  • For every disease, first we compute:
    • number of cured cases = number of cases - number of deaths
  • For each state, these values are normalized by dividing by the state’s total population.
  • Compute rate of increase of number of cases and rate of increase of number of cured cases for year 2010-11
